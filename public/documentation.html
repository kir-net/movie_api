myFlix API Documentation

The objective of this work-in-progress project is to build a server-side component of a “movies” web application called myFlix.
The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

URL endpoints and request methods

Request Endpoint URL HTTP Method Query parameters Request body data format Response body data format
Return list of all movies /movies GET None None JSON object holding data about all movies, e.g. [{"Title": "Down by Law", "Year": 1986, "Genre": "Independent", "Director": "Jim Jarmusch"}, {...}, ...]
Return info about one movie by title /movies/[title] GET :title None JSON object holding data about requested movie, e.g. {"Title": "Down by Law", "Year": 1986, "Genre": "Independent", "Director": "Jim Jarmusch"}
Return info about a genre by name /genres/[Genre] GET :genre None JSON object holding data about requested genre, e.g. "A biographical film or biopic is a film that dramatizes the life of a non-fictional or historically-based person or people. Such films show the life of a historical person and the central character's real name is used."
Return info about a director by name /directors/[name] GET :director None JSON object holding data about requested director, e.g. {"name": "Woody Allen", "birthyear": 1935, "bio": "He began his career writing material for television in the 1950s, mainly Your Show of Shows (1950-1954) working alongside Mel Brooks, Carl Reiner, Larry Gelbart, and Neil Simon. He also published several books featuring short stories and wrote humor pieces for The New Yorker. In the early 1960s, he performed as a stand-up comedian in Greenwich Village. He released three comedy albums during the mid to late 1960s, earning a Grammy Award for Best Comedy Album nomination for his 1964 comedy album entitled simply, Woody Allen."}
Add new user /users POST :users JSON object holding user data, e.g. { "name": "Carlos", "favoriteMovies": ["Down by Law", "Bullets Over Broadway"] } JSON object holding the new user's data, e.g. { "name": "Carlos", "favoriteMovies": [ "Down by Law", "Bullets Over Broadway" ], "id": "0512c1b0-d179-4a88-a4b0-1c02eca255bc" }
Update user name /users/[id]/[name] PUT :id JSON object holding new user's name, e.g. { "name": "Albert" }' JSON object holding the new user's data, e.g. { "id": 1, "name": "Albert", "favoriteMovies": [ "Winter's Bone" ] }
Add a movie to a user's list of favorites /users/[id]/[favoriteMovies] PUT :id, :title None text output confirming requested action, e.g. "Dead Man has been added to your favorite's list."
Remove a movie from a user's list of favorites /users/[id]/[favoriteMovies] DELETE :id, :title None text output confirming requested action, e.g. "Winter's Bone has been removed from your favorite's list."
Delete user /users/[id] DELETE :id None text output confirming requested action, e.g. "User 1 has been removed from database."
